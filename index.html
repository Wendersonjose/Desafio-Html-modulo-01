Hierarquia de Títulos com <h1> a <h6>

<h1> a <h6> são usadas para definir títulos e subtítulos. A tag <h1> é o título mais importante da página e deve ser usada apenas uma vez. As demais seguem uma ordem de importância decrescente até o <h6>.

Parágrafos e Formatação de Texto

<p> é usado para criar parágrafos. Dentro dele, usamos tags como:

  • <strong>: deixa o texto em negrito.
  • <i>: deixa o texto em itálico.
  • <u>: sublinha o texto.
  • <small>: reduz o tamanho do texto.

Também usamos subscrito (<sub>) e sobrescrito (<sup>) para fórmulas como H2O e x2.

Listas Ordenadas e Não Ordenadas

<ol> cria listas ordenadas (numeradas):

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

<ul> cria listas não ordenadas (com marcadores):

  • HTML
  • CSS
  • JavaScript

<li> define cada item da lista, seja ordenada ou não.
